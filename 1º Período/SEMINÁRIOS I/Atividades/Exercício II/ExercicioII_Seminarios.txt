Exercício 1 - 

Para concatenação de arquivos TXT através do comando "cat" é necessário que os arquivos .txt estejam
no mesmo diretório e logo após basta inserir "cat (nome dos arquivos em sequencia) > (nome do arquivo 
que conterá os textos concatenados).
Exemplo.:

cat introducao.txt desenvolvimento.txt conclusao.txt > texto.txt

Exercício 4 -

- ARQUIVO SHELLSCRIPT : 

DIR=$1
echo "$DIR"
if [ -d $DIR ]; then
echo 'diretorio existente'
else
mkdir -p $"DIR/"
chmod 777 -R "$DIR/"
fi
ARQ="texto.txt"
if [ -e "$ARQ" ]; then
echo "arquivo existe. será feita uma cópia"
cp -R "/home/~$/$ARQ" "/home/~$/$DIR/texto.backup.txt"
cd "$DIR"
ls
else
echo "arquivo não existe"
fi

Exercício 5 -

- ARQUIVO SHELLSCRIPT 

if [ $1 = "+" ];
then
   soma=$(($2+$3))
        echo 'SOMA:' $soma
fi

if [ $1 = "-" ];
then
   sub=$(($2-$3))
        echo 'SUBTRACAO:' $sub
fi

if [ $1 = "p" ];
then
   mult=$(($2\*$3))
        echo 'MULTIPLICACAO:' $mult
fi

if [ $1 = "d" ];
then
   div=$(($2/$3))
        echo 'DIVISAO:' $div
fi
